Snap AI is a mobile app that uses Stable Diffusion to turn prompts into ideas.

[![Snap AI](/images/get_it_on_google_play.png)](https://play.google.com/store/apps/details?id=com.lonedreamdstudio.snap_ai)

## Current Features
- Text to Image: Turn prompts into image
- Image to Image: Turn the given image with prompts into another image

## Tech Stack:
- **Flutter**: Used for the mobile app front-end framework.
- **Firebase**: Used for the back-end (Authentication, Firestore, Storage, Cloud Functions, Hosting).
- **NextJS**: Currently used only for user deletion to fulfills user policy. Will be developed in the future into a full landing website for the app.
- **ModelsLab**: A stable diffusion provider.
