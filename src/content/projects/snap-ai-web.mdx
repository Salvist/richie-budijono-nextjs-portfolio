Snap AI Web is a simpler version of the mobile app that uses Stable Diffusion to turn prompts into ideas.

Check the app here: <a href="https://snap-ai-68297.web.app/" target="_blank" rel="noopener">Snap AI Web App</a>

## Current Features
- Text to Image: Turn prompts into image

## Upcoming Features
- Customized parameters for Text to Image
- More models
- Text to Video: Turn prompts into video

## Tech Stack:
- **Flutter**: Used for the web app front-end framework.
    - flutter_bloc
    - go_router
- **Firebase**: Used for the back-end
    - Authentication
    - Firestore
    - Storage
    - Cloud Functions
    - Cloud Scheduler
    - Hosting
- **ModelsLab**: A stable diffusion provider.
