## Tech Stack:
- **Flutter**: Used for the mobile app front-end framework.
    - flutter_bloc
    - flutter_quill
    - shared_preferences
    - sqflite
- **SQLITE**: Used to store notes locally.
- **Quill**: A wonderful rich text editor so the user can style their notes.
